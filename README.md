# VirusJump v2 - Бездна

## Запуск
```
pip install pygame-ce
python main.py
```
(pygame-ce поддерживает Python 3.14)

## Управление
| Клавиша | Действие |
|---------|----------|
| ← →  / A D | Движение |
| Z          | Атака    |
| ESC        | Пауза    |

## Враги — типы вирусов
| Враг | Цвет | Описание |
|------|------|----------|
| ВИРУС | Красный | Заражает файлы |
| ЧЕРВЬ | Зелёный | Распространяется по сети |
| РУТКИТ | Фиолетовый | Скрытый удалённый доступ |
| БОТ | Голубой | Выполняет команды без ведома |
| ВЫМОГАТЕЛЬ | Оранжевый | Шифрует данные |
| РЕКЛАМНОЕ ПО | Жёлтый | Встраивает рекламу |
| ШПИОНСКОЕ ПО | Пурпурный | Крадёт пароли |
